# 基本运算

nml 支持以下运算符：

```bash
算术运算符：+, -, *, /, %
比特运算符：&, |, ^, &&, ||, <<, >>
比较运算符：==, !=, <=, >=, <, >
三元运算符：? :
```

此外，还有一些函数：

```bash
三角函数：sin(),cos(),tan(),asin(),acos(),atan()
平方根：sqrt()
绝对值：abs()
四舍五入：round()
取整：int()
```
